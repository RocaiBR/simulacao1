# Projeto Empresarial

 O Projeto Nilo visa o controle da temperatura e hummidade de forma automatizada.
  Sendo possível seu uso em barracões de armazenamento, salas de escritório, fábricas entre outos locais.

## Problema

Controle da temperatura e humidade do local da instação de maneira remota.

## Publico

Empresas de pequeno e médio porte

## Objetivo

Controle energético
Integração de trabalhadores neurodivergentes
Sustentabilidade
